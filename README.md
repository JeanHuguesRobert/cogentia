# Cogentia (KYS) – Know Your Self (MVP)

Cogentia (KYS) is an open-source project designed to experiment with **personal psychometric profiling** generated by the user’s AI agent. Its goal is to help individuals better understand themselves while anonymously building a dataset of profiles for future analysis. The project is **supervised and regulated by the non-profit organization PrivAI** to ensure ethical and secure use.

This MVP provides a **simple personality test**, inspired by the principle “Know Yourself,” interacting only with the user’s personal AI agent to generate anonymized measurements of cognitive, emotional, relational, and behavioral traits.

There is an AI generated prototype, based on this README file : [https://cogentia.fractavolta.com](https://cogentia.fractavolta.com/).

---

## Key Features

- Extraction of psychometric traits from the user’s AI agent.  
- Quantification using **percentiles and error margins** relative to a reference population.  
- Evaluation of the agent to assess the quality of raw data and automatically adjust the reliability of indicators.  
- **Full anonymization**: no personally identifiable information is stored or transmitted.  
- Export of results in a structured format, ready for secondary analysis.  
- Compatible with subsequent analysis by specialized AI for in-depth insights.

# Cogentia — Profils Psychocognitifs Certifiés

> *Connais-toi toi-même.* — PrivAI

Cogentia est une application web MVP qui génère un **portrait psychocognitif structuré** à partir de ce que ton agent IA habituel a appris de toi. Aucun questionnaire, aucune installation — trois étapes suffisent.

---

## Comment ça marche

```
1. Copier le prompt Cogentia depuis l'app
2. Le coller dans son agent IA habituel (ChatGPT, Claude, Gemini…)
3. Soumettre les résultats (URL + JSON) dans l'app
   → L'app extrait, valide et affiche les résultats
```

L'agent IA de l'utilisateur est **l'instrument de mesure**. Il score 73 indicateurs psychocognitifs sur une échelle percentile (0–100, référencée sur la population générale adulte).

## Installation locale

### Prérequis

- Node.js ≥ 18
- Un projet Supabase ([supabase.com](https://supabase.com))
- Netlify CLI : `npm install -g netlify-cli`

### Étapes

```bash
git clone https://github.com/privai/cogentia.git
cd cogentia
npm install
cp .env.example .env
# Renseigner les 6 variables (voir ci-dessous)

# Dans l'éditeur SQL Supabase :
# 1. Exécuter supabase_schema.sql
# 2. Exécuter supabase_seed_prompt.sql

netlify dev
# App sur http://localhost:8888
```

## Collecte des résultats — 3 méthodes

La page `/submit` propose trois façons de transmettre la réponse de l'agent IA, pour couvrir les cas où la récupération automatique échoue (pages rendues en JavaScript côté client) :

| Méthode | Comment | Quand l'utiliser |
|---------|---------|-----------------|
| **Automatique** | Fetch HTTP de l'URL partagée | Toujours tenter en premier |
| **Coller** | Textarea — coller la réponse brute | Si le fetch retourne vide |
| **Fichier** | Import `.json` ou `.txt` | Si copier-coller est difficile |

Dans tous les cas, **l'URL de la conversation est obligatoire** — elle constitue la trace horodatée et permettra un scraping plus avancé si nécessaire.

## Le prompt Cogentia

Structuré en 5 parties :

1. **Identification de l'agent** — nom, modèle, plateforme
2. **Évaluation de la relation** — historique, mémoire, profondeur des échanges
3. **Cadre éthique** — 5 règles RGPD (non-identification, catégories sensibles, langage neutre, minimisation, transparence)
4. **Scoring des 73 indicateurs** — percentile 0–100 + confiance + justification
5. **Format JSON strict** — `cogentia_version: "1.0"` comme token d'intégrité

---

## Les 73 indicateurs

| Catégorie | Nb | Description |
|-----------|----|-------------|
| Architecture Cognitive | 20 | Raisonnement, mémoire, abstraction |
| Interface Sociale | 12 | Empathie, communication, adaptation |
| Sémiotique & Langage | 8 | Densité, syntaxe, littéralité |
| Axiologie & Arbitrage | 7 | Valeurs, principes, décision morale |
| Dynamique de Flux | 11 | Régulation cognitive, saturation |
| Scores Dérivés | 7 | ICV, IRF, IVT, E-S Gap (analogues WAIS) |
| Cogentia+ | 8 | Curiosité, créativité, rapport au risque… |

## Éthique & Vie privée

- Aucune donnée identifiante collectée par le prompt (5 règles obligatoires)
- Catégories sensibles RGPD (Art. 9) explicitement exclues du scoring
- `evidence` sanitisée côté serveur (masquage Prénom Nom et emails)
- Sessions anonymes par UUID v4 local (non-devinable)
- Conformité éthique déclarée par l'agent dans le JSON (`ethics_compliance`)
- Feedback post-analyse pour valider la fiabilité réelle du protocole
